# Professional IT Portfolio

A modern, dark-themed portfolio website designed for an IT undergraduate.

## Features
- **Modern Design**: Dark aesthetic with neon teal accents.
- **Responsive**: Fully adapts to Mobile, Tablet, and Desktop.
- **Interactive**: Smooth scrolling, mobile menu, and hover effects.
- **Sections**: Hero, About, Skills, Projects, Contact.

## How to Use
1. **Open in Browser**: Simply double-click `index.html` to view the portfolio in your browser.

## Customization
- **Images**: Place your profile picture in the `assets` folder and update the `src` in `index.html`.
- **Colors**: Open `css/style.css` and modify the variables at the top (`:root`) to change the color scheme.
- **Projects**: Edit the "Projects" section in `index.html` to include your actual work.
